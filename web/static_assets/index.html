Beholder

An Encounter Management Tool for DMs

  • Welcome! You can get started by adding a few creatures to your library
  • Beholder is using Open 5E's API to import creatures and spells.
    • If you can't find the resource you're looking for, you can always create it yourself.
  • When you import a creature, Beholder will do its best to import its actions and spells.
    • Beholder is trying its best, but it's far from perfect!
    • You'll want to double check creatures/spells after import and possibly clean them up.
  • You can also create Player Characters - really just placeholders for encounter turn order.
  • After you've created a few creatures, you're ready to create an encounter and add creatures to it!
